# 5. Guia de Uso

Esta seção descreve como um usuário final pode interagir com o **Assistente de Conteúdo com IA** para obter o máximo da ferramenta.

### Passo 1: Inserir a URL do Perfil

Na página inicial, você encontrará um campo de texto principal. Insira a URL completa do perfil de rede social que você deseja analisar.

*Exemplo: `https://www.instagram.com/nomedousuario`*

![Placeholder do campo de inserção de URL](https://via.placeholder.com/700x250.png?text=Campo+para+inserir+a+URL+do+perfil)

Pressione o botão "Analisar". A aplicação começará o processo de coleta e processamento de dados. Isso pode levar alguns instantes.

### Passo 2: Analisar os Gráficos

Assim que a análise for concluída, você será levado a um painel de controle (dashboard) com vários gráficos. Use-os para entender visualmente:
-   **Quais temas** você mais aborda.
-   O **nível de engajamento** de suas postagens recentes.
-   Qual **formato de conteúdo** (imagem, vídeo, etc.) funciona melhor para você.

### Passo 3: Interagir com o Assistente de IA

Abaixo ou ao lado dos gráficos, você encontrará a janela de chat do assistente de IA. O assistente já terá o contexto da análise do seu perfil. Você pode começar a fazer perguntas ou pedir tarefas.

**Exemplos de Prompts:**

-   `"Resuma os principais insights da minha análise."`
-   `"Quais foram os 3 piores posts em termos de engajamento?"`
-   `"Me dê 5 ideias para vídeos curtos (Reels/Shorts) baseados nos meus temas de sucesso."`
-   `"Escreva uma legenda para um post sobre 'produtividade', usando um tom mais informal."`

### Passo 4: Aplicar as Sugestões

Use as ideias, os insights e os rascunhos gerados pela IA para planejar seu calendário de conteúdo. Copie os textos sugeridos e adapte-os conforme necessário antes de postar. O objetivo é usar a IA como um co-piloto criativo, não como um substituto completo.
